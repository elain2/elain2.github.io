## Responsibilities

- Designed and developed full web application as individual final project
- Integrated Flickr API to search and display cat images based on user location
- Implemented Google Map API for location-based content filtering
- Built tag and filter system for stray cat-related content

## About

Recently, terrible crimes against street cats have occurred in Korea. Street cats are driven to animal abuse because they carry germs. However, street cats can reduce their population through management, and if they maintain a clean environment through people's attention, they do not search for trash or carry diseases. With people's attention, street cats will become a small happiness of the day. This web service was planned to share the updates of cats near the residence using the Flickr API.
