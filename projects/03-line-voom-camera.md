## Responsibilities

- Owned VoomCameraViewController — AVFoundation capture session, live preview rendering, and device-specific format selection
- Designed real-time preview effects pipeline using Metal and Core Image
- Architected Storyboard data model with NSCoding and schema versioning for full editing state
- Built VOOMVideoComposer for multi-track video/audio composition using AVMutableComposition
- Ensured thread safety using GCD and OperationQueue; optimized memory for large multi-track edits

## About

Led the iOS team for LINE's short-form video product shipped to global users (JP/TW/TH). Deep expertise in UIKit, AVFoundation, Metal, and asynchronous programming with GCD/OperationQueue. Delivered polished camera UI with real-time effects and robust video editing capabilities.
